  • Javascript(JS)

    • An object-oriented computer programming language commonly used to create interactive web browsers.

  • Operators

    • A symbol that is used to perform an operation. The most commonly operators arearthmetic, but there are others such as comparison operators.

  • Variables

    • A variables purpose is to store information so that it can be used later. A variable is a symbolc name that represents some data that you set.

  • Variable naming conventions

    • The variable's name represents what information the variable contains. They are called variables because the represented information can change but the operations on the variable remain the same.

  • Functions

    • A JavaScript function is a block of code designed to perform a particular task. A JavaScript function is executed when "something" invokes it (calls it).

  • Methods

    • The method attribute specifies how to send form-data (the form-data is sent to the page specified in the action attribute). The form-data can be sent as URL variables (with method="get") or as HTTP post transaction (with method="post").

  • Arguments

    • When a method is called, the arguments are the data you pass into the method's parameters

  • Parameters

    • A parameter is a named variable passed into a function. Parameter variables are used to import arguments into functions.

  • Return

    • The return statement stops the execution of a function and returns a value from that function.

  • Chaining methods

    • Chaining is based on task analysis, whereby sub-behaviors are recognized as requirements for task mastery. DESCRIPTION. Chaining breaks a task down into small steps and then teaches each step within the sequence by itself.

  • Strings

    • A string is a data type used in programming, such as an integer and floating point unit, but is used to represent text rather than numbers. It is comprised of a set of characters that can also contain spaces and numbers. For example, the word "hamburger" and the phrase "I ate 3 hamburgers" are both strings.
